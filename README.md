# Gym App Maqueta

Maqueta funcional para presentar a un cliente en la primera entrevista.

## Tecnologías
- HTML
- CSS
- Bootstrap
- Firebase Auth
- Firestore
- Firebase Hosting

## Secciones
- Login
- Dashboard
- Perfil del cliente
- Rutinas
- Turnos
- Tienda
- Pagos

## Usuarios demo
- admin@gym.com
- trainer@gym.com
- cliente@gym.com

## Objetivo
Mostrar una versión navegable del sistema para validar funcionalidades y diseño con el cliente.
